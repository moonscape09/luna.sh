## Prerequisites:
- Ensure Conda is installed on your system.

## Setup Instructions:
1. Create the Conda environment from `luna-environment.yml`:
   ```bash
   conda env create -f luna-environment.yml
   ```
2. Activate the Conda environment
  ```bash
  conda activate luna-env
  ```
3. Launch `luna.sh`
  ```bash
  source launch.sh
  ```
